# Домашнее задание 7 — кратчайший путь (алгоритм Дейкстры)

Формат ввода (stdin):

```
n m s t
u1 v1 w1
u2 v2 w2
...
um vm wm
```

- `n` — число вершин (0..n-1)
- `m` — число рёбер
- `s` — стартовая вершина
- `t` — целевая вершина
- далее `m` строк: ориентированное ребро `u -> v` с неотрицательным весом `w` (целым)

Вывод: одно число — длина кратчайшего пути `s -> t`, либо `INF` если пути нет.

## Пример
```
Ввод:
4 4 0 3
0 1 1
1 2 2
0 2 5
2 3 3

Вывод:
6
```

## Сборка / запуск (Maven)

```bash
mvn -q -DskipTests=false test
mvn -q -DskipTests package
java -jar target/hw7-1.0-SNAPSHOT.jar < input.txt
```
